```

🔑 비밀번호: master123!
👤 이름: 마스터 교사
```

4. **"회원가입"** 버튼 클릭

## ⚠️ 중요 사항

### MongoDB 필요
- 회원가입 및 로그인 기능을 사용하려면 **MongoDB가 실행 중**이어야 합니다.
- MongoDB가 없으면 데이터베이스에 저장되지 않습니다.

### MongoDB 실행 방법

#### 옵션 1: 로컬 MongoDB
```powershell
# MongoDB 서비스 시작
net start MongoDB

# 또는 직접 실행
mongod
```

#### 옵션 2: MongoDB Atlas (클라우드 - 무료)
1. https://www.mongodb.com/cloud/atlas 접속
2. 무료 계정 생성
3. 클러스터 생성 후 연결 문자열 복사
4. `server/.env` 파일에 `MONGODB_URI` 설정

---

## 🔧 MongoDB 없이 테스트하기

MongoDB 없이도 UI는 확인할 수 있지만, 실제 데이터 저장은 되지 않습니다.

1. 브라우저에서 `http://localhost:5173` 접속
2. 회원가입 페이지 확인 (MongoDB 없으면 저장 실패)
3. UI 및 기능 확인
